# concert-hall-website
This is my Concert Hall website related to the "Event Center Website" assignment in the HTML and CSS module from Coding Temple.

This website was coded via Visual Studio Code. It includes:

• index.html — Home page (introduction to the event center and featured events).
• about.html — About the venue (history, mission, team).
• events.html — Event listings or schedule.

And it also includes special features such as:

• A navigation menu linking all pages together.
• Membership pricing and features table.
• Events calendar.
• A contact form with text inputs, email input, textarea, and a submit button for inquiries or bookings.
• A footer with social media links or copyright information.
• Hover and focus effects for interactive elements (buttons, links, images).
• A responsive layout that adapts to mobile and desktop screens.
• Favicon designed by Zava (azthomaz).
• “Back to top” button.

To be able to view all features coded, you may need a source code editor with Preview extensions (eg.: Live Server) to open the code in a browser of preference and have a better experience reviewing my code.

The biggest challenge for me was coding the "about.html" page and organising the elements in a visually appealing and cohesive way. The membership pricing and features table was the cherry on top in terms of complexity (as it was my first HTML & CSS coded table).

I hope you enjoy my Concert Hall website!
